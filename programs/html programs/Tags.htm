Text Formatting tags

bold tag

b tag is used to apply bold on the text.
Example:bold text

italic tag

i tag is used to apply italic on the text.
Example:italic text

strong tag

strong tag is used to represent the text in strong.
Example:strong text

underline tag

u tag is used to apply underline on the text.
Example:underlined text

del tag

del tag is used to display the deleted text.
Example:demo text

mark tag

Mark tag is used to mark the text.
Example:demo on mark text

center tag

center tag is used to align text to center.
Example:

text to the center

strike tag

strike tag is used to strike the text.
Example:strike text

teletype tag

tt tag is used to display the text.
Example:teletype text demo

big tag

big tag is used to display the text larger than usual.
Example: text in larger size

small tag

small tag is used to display the text in smaller size.
Example:text in small size

qutation tag

qutation tag is used to apply qutations on the text.
Example: text inside qutation

sub tag

sub tag is used to display text as subscript
Example:calcium carbonate caco3

superscript tag

sup tag is used to display text as superscript
Example:(a+b)2=a2+b2+2ab

insert tag

ins tag is used to add text , usually it is used after del tag
Example:failure analysing failure is stepping stone of success

HTML attributes are special words used inside the opening tag to control the element's behaviour. HTML attributes are a modifier of an HTML element type. An attribute either modifies the default functionality of an element type or provides functionality to certain element types unable to function correctly without them. In HTML syntax, an attribute is added to an HTML start tag.
